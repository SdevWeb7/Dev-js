<h2>Cheat Sheet</h2>

<p>
  Maitriser sont IDE est un art. Commencez par quelques raccourcis clés et
  intégrez en des nouveaux progressivement. La liste suivante représente les
  commandes par défaults, il en existe beaucoup d'autres, n'hésitez pas a
  changer l'attribution des raccourcis en fonction de vos préférences.
</p>

<h3>Navigation</h3>

<ul>
  <li>Ctrl + P : Accéder à un fichier par son nom</li>
  <li>Ctrl + Shift + P : Ouvre la palette de commandes</li>
  <li>Ctrl + O : Ouvre un fichier</li>
  <li>Ctrl + K + O : Ouvre un dossier</li>
  <li>Ctrl + G : Aller à une ligne spécifique de votre fichier</li>
</ul>

<h3>Recherche</h3>
<ul>
  <li>Ctrl + F : Rechercher dans le fichier</li>
  <li>Ctrl + Shift + F : Rechercher dans tous les fichiers</li>
  <li>Ctrl + H : Remplacer dans le fichier</li>
  <li>Ctrl + Shift + H : Remplacer dans tous les fichiers</li>
</ul>

<h3>Edition</h3>
<ul>
  <li>Ctrl + Space : afficher les suggestions de code</li>
  <li>Ctrl + flèche haut ou bas : aller à la ligne précédente ou suivante</li>
  <li>Ctrl + L : sélectionner la ligne</li>
  <li>Ctrl + / : commenter le code</li>
</ul>

<h3>Terminal et console</h3>
<ul>
  <li>Ctrl + J : ouvrir le terminal et la console.</li>
  <li>Ctrl + C : terminer une tâche en cours.</li>
</ul>
