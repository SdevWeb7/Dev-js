<h2>Prendre en main VS Code</h2>

<h3>2.1 Installation et configuration initiale</h3>
<p>
  VS Code est compatible avec différents systèmes d'exploitation tels que
  Windows, macOS et Linux. Pour profiter de ses puissantes fonctionnalités, vous
  pouvez suivre les instructions ci-dessous pour installer l'éditeur de texte su
  votre système d'exploitation.
</p>
<ul>
  <li>
    Cliquez sur <b>Downaload for Windows</b> après vous être rendu à l'adresse :
    <a href="https://code.visualstudio.com/" target="_blank">
      Télécharger VS Code
    </a>
  </li>
</ul>

<h3>2.2 Configuration de VS Code</h3>
<p>
  Une fois installé, vous pouvez configurer VS Code pour qu'il s'adapte à vos
  préférences. Voici quelques éléments à configurer :
</p>
<ul>
  <li>Thème</li>
  <li>Langue</li>
  <li>Formatage</li>
</ul>
