<h2>1.1 Qu'est-ce que Visual Studio Code ?</h2>
<p>
  VS Code est un éditeur de code open source développé par Microsoft.
  Contrairement à un Environnement de Dévezloppement Intégré (IDE) traditionnel,
  qui comprend un large évantail d'outils dédiés au développement logiciel
  complet tel que le débogage avancé, un <b>éditeur de code</b> comme VS Code se
  concentre sur l'écriture et la modification de code. Néanmoins, il est très
  apprécié par les développeur web à travers le monde part sa grande variété de
  plugins et extensions. Bien configuré, il offre des fonctionnalités comparable
  à celle d'un IDE.
</p>

<h3>Quelques fonctionnalités :</h3>
<ul>
  <li>Intégration de Git</li>
  <li>Support de l'autocomplétion</li>
  <li>Support de l'analyse statique</li>
  <li>Support de la refactorisation</li>
  <li>Support de la navigation dans les fichiers</li>
  <li>Support des extensions</li>

<h3>Pourquoi utiliser Visual Studio Code ?</h3>
<p>
  Conçu pour être rapide, léger et hautement configurable, VSCode plait aux
  développeurs de tous niveaux. Il est également très facile à apprendre et à
  utiliser, et ne vous coutera rien, oui oui c'est gratuit.
</p>
