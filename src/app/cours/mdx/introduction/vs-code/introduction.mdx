<h1>Introduction à Visual Studio Code</h1>

<h2>1.1 Qu'est-ce que Visual Studio Code ?</h2>
<p>
  Visual Studio Code est un éditeur de texte open source développé par
  Microsoft.qui permet de créer et de modifier du code. Apprécié par les
  développeurs web car il prend en charge de nombreux langages de programmation
  comme JavaScript, Python, PHP, Java et bien d'autres.
</p>

<h3>Quelques fonctionnalités :</h3>
<ul>
  <li>Intégration de Git</li>
  <li>Support de l'autocomplétion</li>
  <li>Support de l'analyse statique</li>
  <li>Support de la refactorisation</li>
  <li>Support de la navigation dans les fichiers</li>
  <li>Support des extensions</li>

<h3>Pourquoi utiliser Visual Studio Code ?</h3>
<p>
  Conçu pour être rapide, léger et hautement configurable, VSCode plait aux
  développeurs de tous niveaux. Il est également très facile à apprendre et à
  utiliser, et offre une grande variété d'extensions pour ajouter des
  fonctionnalités supplémentaires gratuitement.
</p>
