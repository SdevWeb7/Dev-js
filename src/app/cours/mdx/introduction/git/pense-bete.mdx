<h1>Git - Pense-Bête</h1>

- **Récupérer un dépôt** : `git clone <url>`
- **Créer un dépôt** : `git init`
- **Ajouter un fichier** : `git add <fichier>`
- **Valider un fichier** : `git commit -m "Message"`
- **Envoyer sur un dépôt** : `git push`
- **Récupérer d'un dépôt fetch+merge** : `git pull`
- **Créer une branche** : `git branch <nom>`
- **Changer de branche** : `git checkout <nom>`
- **Fusionner une branche** : `git merge <nom>`
- **Supprimer une branche** : `git branch -d <nom>`
- **Voir les modifications** : `git status`
- **Voir l'historique** : `git log`
- **Annuler les modifications** : `git checkout -- <fichier>`
- **Récupérer les modifications** : `git fetch` puis `git merge origin/master`
