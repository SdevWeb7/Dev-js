<h1>Git - Exercices</h1>

<p>1) Pour s'habituer aux manipulations de Git :</p>
<ul>
   <li>Créer un dépôt localement</li>
   <li>Créer un dépôt sur une plateforme de gestion (Github par exemple)</li>
   <li>Les lier</li>
   <li>Faire des modifications et push</li>
</ul>




<p>2) Pour apprendre à gérer les conflits, avec un ou plusieurs collègues, essayez de travailler sur le meme dépôt distant en meme temps. Vous pouvez ensuite essayer le mécanisme des "pull requests"</p>
