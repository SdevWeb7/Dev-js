## Créer un projet Next.js :
```bash
npx create-next-app nomduprojet
```

- Tester toutes les informations reçues, familiarisez-vous tranquillement avec Next.js.


- Prenez le temps de vous familiariser avec les concepts de base de Next.js.


- Essayez de “deploy” votre application gratuitement grâce à Vercel qui propose de mettre en production votre application gratuitement en connectant votre Dépot Github.


- Le processus de déploiement d'une application sans fonctionnalité avancé est plutôt simple. Je vous laisse découvrir Vercel par vous-même.
