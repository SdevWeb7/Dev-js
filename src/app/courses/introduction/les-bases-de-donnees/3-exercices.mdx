- Installer WAMP. Télécharger WAMP ou Clubic.com
- Créer une base de donnée avec phpmyadmin
- Créer 2 table avec une relation entre elles (il faudra aller dans l'onglet Vue relationnel)
- Insérer des données dans ces tables

### Se familiariser avec les 4 types de requêtes SQL (Select, Insert, Update, Delete) en effectuant des requêtes à l'aide de phpmyadmin
