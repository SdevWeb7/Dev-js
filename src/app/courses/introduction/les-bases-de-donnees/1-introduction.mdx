Une base de données est un ensemble de données organisées et structurées.

Ces données sont stockées à l'aide d'un système de gestion de base de données (SGBD) qui permet de les manipuler.

Il existe plusieurs types de bases de données, en développement informatique, les plus courantes sont les bases de données relationnelles.

Cependant, il existe d'autres types de bases de données, comme les bases de données NoSQL, les bases de données orientées graphe, les bases de données en mémoire, les bases de données distribuées, les bases de données temps réel, les bases de données multi-modèles, les bases de données géospatiales, etc.

Ce cours a pour but de comprendre la structure, le stockage et la récupération de données depuis une base de donnée dans le développement web donc nous allons aborder les bases de données relationnelles.
