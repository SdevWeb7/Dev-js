## Pense-bête

### Une requête/réponse HTTP :
- se fait entre deux ordinateurs
- possède une méthode (GET par défaut)
- contient des headers (informations sur la requête/réponse)
- peut contenir un body (des données souvent au format JSON)
- envoi un cookie pouvant contenir une clé d'authentification
