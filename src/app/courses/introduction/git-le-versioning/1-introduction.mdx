Git est un logiciel de gestion de versions décentralisé.

C'est un outil incontournable pour les développeurs. Il permet de suivre l'évolution du code source d'un projet et de collaborer avec d'autres développeurs.

Les fonctionnalités de Git sont nombreuses et permettent de revenir en arrière en cas de problème, de gérer efficacement les branches, les conflits, les tags, les commits, etc.

[![git branches](/git/branche_git.png)](/git/branche_git.png)


Utiliser Git est une compétence essentielle pour tout développeur cependant il peut être difficile à prendre en main. C'est pourquoi nous allons voir dans ce cours les bases de Git et comment l'utiliser au quotidien.
