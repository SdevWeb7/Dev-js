## Pour s'habituer aux manipulations de Git :

- Créer un dépôt localement
- Créer un dépôt sur une plateforme de gestion (Github par exemple)
- Les lier
- Faire des modifications et push


### Pour apprendre à gérer les conflits, avec un ou plusieurs collègues, essayez de travailler sur le meme dépôt distant en meme temps. Vous pouvez ensuite vous entrainer au mécanisme des pull requests
