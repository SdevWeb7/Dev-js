## Qu'est-ce que Visual Studio Code ?
VS Code est un éditeur de code open source développé par Microsoft.

Contrairement à un Environnement de Développement Intégré (IDE) traditionnel, qui comprend un large éventail d'outils dédiés au développement logiciel complet tel que le débogage avancé, un éditeur de code comme VS Code se concentre sur l'écriture et la modification de code. Néanmoins, il est très apprécié par les développeurs web à travers le monde part sa grande variété de plugins et extensions. Bien configuré, il offre des fonctionnalités comparables à celle d'un IDE.

Quelques fonctionnalités :
- Intégration de Git
- Support de l'autocomplétion
- Support de l'analyse statique
- Support de la refactorisation
- Support de la navigation dans les fichiers
- Support des extensions


## Pourquoi utiliser Visual Studio Code ?
Conçu pour être rapide, léger et hautement configurable, VSCode plait aux développeurs de tous niveaux. Il est également très facile à apprendre et à utiliser, et ne vous coutera rien, oui oui c'est gratuit.
