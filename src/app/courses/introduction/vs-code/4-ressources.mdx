## Pense-bête

Maitriser sont IDE est un art. Commencez par quelques raccourcis clés et intégrez en des nouveaux progressivement. La liste suivante représente les commandes par défaults, il en existe beaucoup d'autres, n'hésitez pas à changer l'attribution des raccourcis en fonction de vos préférences.

### Navigation
- Ctrl + P : Accéder à un fichier par son nom
- Ctrl + Shift + P : Ouvre la palette de commandes
- Ctrl + O : Ouvre un fichier
- Ctrl + K + O : Ouvre un dossier
- Ctrl + G : Aller à une ligne spécifique de votre fichier


### Recherche
- Ctrl + F : Rechercher dans le fichier
- Ctrl + Shift + F : Rechercher dans tous les fichiers
- Ctrl + H : Remplacer dans le fichier
- Ctrl + Shift + H : Remplacer dans tous les fichiers


### Edition
- Ctrl + Space : afficher les suggestions de code
- Ctrl + flèche haut ou bas : aller à la ligne précédente ou suivante
- Ctrl + L : sélectionner la ligne
- Ctrl + / : commenter le code


### Terminal et console
- Ctrl + J : ouvrir le terminal et la console.
- Ctrl + C : terminer une tâche en cours.
