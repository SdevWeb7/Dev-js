## Installation et configuration initiale
VS Code est compatible avec différents systèmes d'exploitation tels que Windows, macOS et Linux. Pour profiter de ses puissantes fonctionnalités, vous pouvez suivre les instructions ci-dessous pour installer l'éditeur de texte sur votre système d'exploitation.


Cliquez sur Download for Windows après vous être rendu à l'adresse: [site officiel VS Code](https://code.visualstudio.com/)


Une fois installé et lancé, vous pouvez configurer VS Code pour qu'il s'adapte à vos préférences. Voici quelques éléments à configurer :

- Thème
- Langue
- Formatage




## Naviguer dans l'interface
Une bonne maitrise de l'interface vous permettra de travailler plus rapidement et de manière plus efficace. L'explorateur de fichier par exemple, vous permet de naviguer dans votre projet, de voir les fichiers, de les ouvrir, de les créer, de les supprimer, etc.

Dans la barre latérale, vous trouverez les icônes de vos extensions, vous pouvez aussi customiser ceci.
